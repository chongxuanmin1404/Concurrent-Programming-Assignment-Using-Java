# Concurrent-Programming-Assignment-Using-Java
A short program that creates a simulation in an airport written entirely in Java. It coordinate landings, docking and takeoffs using concurrency concepts. 
The program simulates the daily processes and normal operations at an airport, planes land and dock to the specified hangar and passengers disembark. Maintenance and refueling of the planes are conducted sequentially and passengers embark. Once all processes are completed the planes receive permission from the ATC (Air Traffic Control) to takeoff.

Tools and Software Used:
Apache Netbeans IDE 12.2

Installation Guide:
1. Download the latest version of Apache Netbeans
2. Create a new Java Application using Ant
3. Import the given files into the default source package
4. Run the program
